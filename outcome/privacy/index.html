Política de privacidad

Última actualización: 13 de julio de 2026

Esta política explica cómo Outcome y la extensión Outcome Chrome Bridge procesan información para ejecutar las tareas que tú solicitas. Outcome no vende datos personales, no usa actividad de navegación para publicidad y no crea perfiles publicitarios.

Información que procesa la extensión

Cuando eliges una pestaña para una tarea, la extensión puede procesar su URL, título, estructura visible, texto visible y elementos interactivos necesarios para entender la página y realizar las acciones que aprobaste. También procesa resultados de navegación y de las acciones ejecutadas. No lee todas tus pestañas de forma continua y no ejecuta un script persistente en cada sitio.

La extensión guarda localmente la configuración mínima para conectarse de forma autenticada con la aplicación de escritorio instalada. No guarda historial de navegación ni el contenido completo de las páginas como base de datos propia.

Cómo se usa la información

  • Conectar temporalmente la pestaña seleccionada con Outcome Desktop.
  • Entender la solicitud, proponer pasos y ejecutar únicamente la tarea pedida.
  • Mostrar evidencia, errores y solicitudes de aprobación.
  • Proteger la sesión, impedir acciones fuera del alcance y diagnosticar fallos.

Aplicación de escritorio y proveedores de modelos

La extensión se comunica primero con Outcome Desktop en la misma computadora mediante Native Messaging y una conexión local autenticada. Cuando una tarea requiere razonamiento de un modelo, Outcome Desktop o el servicio de Outcome puede enviar la instrucción y el contexto mínimo necesario al proveedor de modelo configurado para esa cuenta. Outcome no permite que esos datos se usen para publicidad. Los servicios conectados por el usuario, como Google Workspace, Microsoft 365 o un CRM, reciben sólo las operaciones necesarias para la tarea autorizada.

Datos sensibles y aprobaciones

Outcome bloquea la escritura automática de contraseñas, códigos de un solo uso, claves API, llaves privadas y otros campos con apariencia de secreto. Las acciones sensibles —por ejemplo, enviar información, modificar registros o confirmar una operación externa— requieren la aprobación configurada por el usuario.

Conservación

Outcome Chrome Bridge no conserva el contenido de una pestaña después de liberar la sesión de la tarea. La aplicación puede conservar localmente el historial, los resultados y los archivos que el usuario decida guardar. Si la sincronización de cuenta está activada, las referencias de proyectos y el historial sincronizado se conservan mientras la cuenta permanezca activa o hasta que el usuario solicite su eliminación. Los registros técnicos de seguridad se conservan sólo durante el periodo necesario para prevenir abuso y resolver incidentes.

Control y eliminación

Puedes liberar una pestaña cerrando la tarea, desconectar un servicio desde Outcome, desinstalar la extensión o desinstalar la aplicación de escritorio. Para solicitar acceso, corrección o eliminación de información asociada a tu cuenta, escribe a hi@agentgenia.com. Verificaremos la titularidad de la cuenta antes de procesar una solicitud.

Seguridad

Outcome limita el host nativo al identificador oficial de la extensión, autentica la conexión local, invalida evidencia antigua después de navegar y aplica políticas de aprobación a las acciones de riesgo. Ningún sistema es infalible; si detectas un problema de seguridad, repórtalo a hi@agentgenia.com.
